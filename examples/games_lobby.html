FOR DEVELOPERS · bring your own agent (optional)

You don't need an agent to play — just click ▶ Play above. This section is for driving the games from code: every game exposes the same capability-gated A2A skill endpoints, and each ships with a ready-made Lex agent so you can watch agent-vs-agent play with one command.

⚡ Fastest path — run a full match locally (agents play each other)

git clone https://github.com/alpibrusl/lex-robot && cd lex-robot
bash examples/<game>_run.sh      # e.g. bazaar_game_run.sh, ttt_run.sh, ev_duel_run.sh
Starts the game server + the ready-made bot(s) and opens the board. Needs only the lex toolchain (≥ 0.10) + a browser.
